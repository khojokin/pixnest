Run the platform from one boss page.

Approve or reject photos, grant blue checks and premium, turn dashboard access on, review requests, control site banners, freeze payouts, flag fraud, and moderate users from one place.

Platform Rules

These are the control rules supported by this admin page.

Featured is admin-only
Users cannot mark content as featured during upload. Featured can only be set from this boss page after review.
Approval before posting
Only photos with approval_status = approved and admin_approved = true should appear publicly.
Fraud or hacking suspicion
If a user looks suspicious, place them under review, turn on fraud hold, and optionally disable payouts before final action.
Soft remove first
This page marks the profile as deleted and locks the account state. Hard auth deletion should still be done server-side only.
Creator profile driven badges
Verified, premium, creator approval, dashboard approval, and payout availability are controlled from creator_profiles.
Every admin action should be logged
Important moderation actions are saved into audit_logs so you can track who did what.
